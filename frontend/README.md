# Music Spotify

## Descripción
Desarrollo y simulación de la interfaz de Music Spotify.

## Características Principales
- Repositorio.
- Facil de usar.
- Colaborativo.

## Tecnologías
### Frontend
- HTML5, CSS3, Boostrap


### Backend 
- Node.js / Javascript / Express
- Base de datos: PosgreSQL

## Estructura del Proyecto
Se considero la estructura del frontend y backend en el mismo repositorio ya que el proyecto trabajado es de caracter individual, archivos de proceso en la raiz del directorio, asignacion de frontend y backend en sus respectivas carpetas.

 - docs/ - Documentación de sprints y retrospectivas
 - src/backend/ - API y lógica de negocio
 - src/frontend/ - Interfaz de usuario
 - src/test/ - Pruebas automatizadas
